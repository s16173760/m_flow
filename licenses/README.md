# Third-Party Licenses

This directory contains license texts for third-party open-source software
redistributed within M-flow.

For the full attribution notice, see [`NOTICE`](/NOTICE)
and [`NOTICE.md`](/NOTICE.md) in the repository root.

## Adding a New License

When vendoring or bundling a new dependency:

1. Copy its license file here, named `<package>-LICENSE` (e.g. `lancedb-LICENSE`).
2. Add an entry to `NOTICE.md` describing the package and its license type.
